# Personal Link Profile

A clean, animated personal link hub built with React, TypeScript, Framer Motion, and Tailwind CSS.

Designed and built by Jose Armando Arellano Ferrara.

## Tech Stack

- React
- TypeScript
- Framer Motion
- Tailwind CSS
- Vite

## Features

- Smooth entrance animations
- Staggered link reveal
- Icon-based navigation
- Responsive layout
- Clean, modern UI
- Custom styling (no templates)

## Purpose

This project serves as a personal link hub and lightweight portfolio entry point.  
Instead of using third-party link aggregators, this page is fully custom-built and deployed.
